오늘 배운 내용을 정리해볼께.
1. 우선 ollama 로컬 서버만들기 
OLLAMA_HOST=0.0.0.0:11434 ollama serve
2. 에이전트 소통 테스트하기 
curl http://localhost:11434/api/generate -d '{
  "model": "gemma4:e4b",
  "prompt": "gd는 제이는 누구야?",
  "stream": false
}'
3. 안티그래비티에 제이슨 파일 만들어 젬마 4 심기
antigravity.config.json
{
  "models": {
    "local": {
      "provider": "openai-compatible",
      "baseUrl": "http://localhost:11434/v1",
      "model": "gemma4:e4b",
      "apiKey": "ollama",
      "temperature": 0.1,
      "maxTokens": 4096,
      "contextWindow": 128000
    }
  },
  "defaultModel": "local",
  "fallback": {
    "enabled": true,
    "model": "claude-opus-4-6",
    "trigger": "context_exceeded"
  }
}
